# Employee Management System

## Description
This project is built using Python.
The Employee Management System is a Python-based application designed to efficiently manage employee records using an SQLite database. 
It provides core functionalities such as adding, viewing, updating, and deleting employee details through a simple command-line interface. 
The project demonstrates the implementation of CRUD operations, database connectivity, and basic data management using Python.

## Features
- Add Employee Details
- View Employee Details
- Update Employee
- Delete Employee details 
- Exit

## Tech Used
- Python
- SQLite3

## How to Run
1. Install requirements
2. Run app.py
